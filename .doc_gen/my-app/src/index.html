SoS Code Example Metadata GUI editor

This app enables you to quickly and easily add the required metadata to publish examples to official AWS documentation.

To get started:

  1. Complete prerequisites on README.
  2. Clone aws-doc-sdk-examples GitHub repo locally. Instructions here.
  3. Open your CLI tool, and navigate to "./doc_gen/metadata/sos_editor/
  4. Enter 'npm install'
  5. Enter 'node run.js' and follow the instructions.
  6. Open this file in your localhost.
